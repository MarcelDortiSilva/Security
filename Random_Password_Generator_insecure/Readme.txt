Password Generator 1.0 - by Marcel Dorti Silva - 2023

This password generator allows you to generate your password taking the UTC Time as source of
entropy - low entropy - it is not recommend for secure password generations, it is the first
step of a series of snippets to teach about about password secure generation.

Use it only for didatic purposes and maybe some websites - it is an alternative to using
Web programs, with this program, you have all the trust of the password generated.
